.. Welcome to Sakura's documentation!
   ====================================

.. image:: images/banner.png
   :scale: 100 %
   :alt: Sakura banner

Sakura [#sakura_url]_
is a data explorer for data generated at the XAS beamline at the Australian Synchrotron.

- Reads step mode and mapping mode data.
- Exports columnar ASCII.
- Cross-platform. Runs on Windows/Mac OSX/Linux
- Released under the Modified BSD-license
